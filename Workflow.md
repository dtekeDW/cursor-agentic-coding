## Workflow

1. update `output/Workshop.md` first
2. update all affected section files in `output/sections/*/section.md`
3. ensure all updated files include navigation links (`Workshop`, `Prev`, `Next`)
4. ensure time limits still hold (`45-50` minutes talk + separate `10` minutes Q&A)
5. check didactic clarity and content correctness
6. bump version and timestamp in `output/Workshop.md`
7. update `README.md` if necessary
8. append short entry to `output/changelog.md` (date, version, file, change)
