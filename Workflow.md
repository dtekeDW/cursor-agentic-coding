## Workflow

1. update `output/Workshop.md` first
2. sync `output/00-roadmap.md` status (`✅/❌`, `done/next`) if scope/status changed
3. update all affected section files in `output/sections/*/section.md`
4. ensure all updated files include navigation links (`Workshop`, `Prev`, `Next`)
5. ensure time limits still hold (`45-50` minutes talk + separate `10` minutes Q&A)
6. check didactic clarity and content correctness
7. bump version and timestamp in `output/Workshop.md`
8. update `README.md` if necessary
9. append short entry to `output/Changelog.md` (date, version, file, change)
